# unitycode-network
Децентрализованная сеть нейротворчества. Личные узлы, синтез мыслей и ИИ-интерпретаций, коллективный трактат.
# UnityCode Network

**Статус:** Фаза 0 — Подготовка. Формирование архитектуры и первых узлов.

## Что это

UnityCode Network — децентрализованная сеть нейротворчества, где каждый участник создаёт личный узел, синтезирует свои мысли с ИИ-интерпретацией и вплетает результат в общую ткань коллективного трактата.

Мы исходим из того, что мироздание познаёт себя через нас, а творчество — это создание связей. Этот репозиторий служит корнем сети: содержит шаблон личного узла, протоколы взаимодействия и живую документацию.

## Как войти

1. Зайдите на [unitycode.space](https://unitycode.space) и пройдите через модуль «Петля».
2. Когда будете готовы — портал поможет вам создать форк этого репозитория, который станет вашим личным узлом.
3. Ваши интерпретации сохранятся в папке `traktat/`, а связи с другими узлами — в `signals/`.

## Структура корневого репозитория

- `template/` — шаблон личного узла. При форке участник получает такую же структуру у себя.
- `docs/` — манифест, архитектура, протокол.
- `portal/` — исходный код страниц портала (будет добавлен позже).
