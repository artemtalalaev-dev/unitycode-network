
# Мой узел UnityCode

Это мой личный узел сети UnityCode. Здесь я храню свои «сшивки» — результаты синтеза моих мыслей и ИИ-интерпретаций.

## Структура

- `traktat/` — мои сшитые тексты.
- `signals/` — ссылки на другие узлы, с которыми я создал связь.
- `config.yml` — настройки узла.
